## 6. 标杆企业记录

建议将 `赣锋锂电 / 赣锋储能` 记录为：

- **标杆企业等级**：`A-`
- **角色定位**：`未来业务合作伙伴 + 方案参考对象 + 供应链/系统集成对标对象`

### 6.1 建议纳入标杆库的原因

1. **产业链完整**
   - 不是单纯做 Pack 或集成，而是具备上游锂资源、材料、电芯、系统、能源服务、回收利用的系统能力。

2. **出海与认证能力较强**
   - 本地资料已经体现出较完整的国际认证与运输资料体系，这对未来做海外业务或日本市场合规非常重要。

3. **系统级交付能力较强**
   - 从公开信息看，其不仅卖电芯，也在做工商业储能和大储系统，具备项目交付和工程化能力。

4. **技术路线有延展性**
   - 既有当前主流的磷酸铁锂储能方案，又在大电芯和固态电池方向继续投入。

### 6.2 对 openBattery 的参考价值

对本项目更有参考意义的，不是直接照搬其整套产品，而是提炼以下能力：

1. **认证包思维**
   - 未来我们的系统资料不应只停留在规格书，而要逐步形成：
   - 产品规格书
   - 安规/EMC
   - 热失控测试
   - 运输测试
   - MSDS
   - 项目交付资料

2. **系统级产品化表达**
   - 赣锋公开材料通常不是只讲“电芯参数”，而是把“电芯 + 柜体/舱体 + EMS/运维 + 场景方案”一起输出。

3. **场景化矩阵思路**
   - 发电侧、电网侧、工商业、户储分层清晰。
   - 这对 `openBattery` 未来从移动电源/BMS 平台延展到更广储能平台时有借鉴意义。

4. **交付与认证先行**
   - 从本地资料看，其工程化、测试、认证、运输适配并行推进，说明成熟厂商不是先把产品堆出来再补资料，而是同步做准入准备。
