## 1. 当前必要结论

1. **日本储能准入判断不能仅以 `PSE` 为核心。**  
   当前更稳妥的判断框架应区分：
   - 国际安全与交付基线：`IEC 62619`、`IEC 62477-1`、`IEC 62933-5-2`、`EMC`、`UN38.3`、`SDS`
   - 日本本地落地主线：`JET / JIS / 系統連系 / JC-STAR / NITE`

2. **AC 输出便携式储能电源当前不按电安法受规制的 `モバイルバッテリー` 处理。**  
   依据当前 `METI` FAQ 口径，可输出 `交流100V` 的 `ポータブル電源` 属于非対象，因此整机层面不宜直接套用 `圆形 PSE`。
   当前更合适的日本落地路线，应理解为 `METI 中间要求事项 + S-JET`；其中 `S-JET` 虽非法定强制认证，但对渠道沟通、采购解释和风险控制具有明显现实价值。

3. **圆形 `PSE` 的必要性必须按产品主功能和流通形态判断。**  
   当前可归纳为：
   - 需要 `圆形 PSE`：以外接 `DC` 供电为主的 `モバイルバッテリー`，以及可单独流通的受规制 `リチウムイオン蓄電池`
   - 不以整机 `PSE` 为主线：以 `AC` 输出为主的便携式储能电源、户储、小型工商业储能、定置大储系统

4. **日本法人品牌 + 中国供应链制造的模式是可行的，但责任必须前置固化。**  
   当前模式下：
   - 日本法人承担市场准入、标签、召回、对外销售与品牌责任
   - 中国工厂承担制造一致性、工厂审核、样机代表性和生产变更控制责任

5. **产品分型先于认证堆叠。**  
   当前建议固定分为四类：
   - `A`：AC 输出便携式储能电源
   - `B`：DC 外付け电源 / モバイルバッテリー
   - `C`：低压并网型储能系统（户储/小型工商业）
   - `D`：大型定置储能系统（柜体/集装箱/公共基础设施）

### 1.1 需进一步确认的项目

1. 便携式产品中是否存在 `可单独流通的电池包`、`外置DC电源部件` 或其他可能触发 `PSE` 的部件级对象。
2. 户储/小工商储产品的 `并网边界`、`PCS 角色边界`、`通信网关边界` 是否已经在规格书中冻结。
3. 日本品牌销售型号、铭牌型号、认证送样型号、ODM/OEM 内部型号是否已完成一对一映射。
4. 中国工厂是否已具备接受 `JET / UL / TÜV` 工厂审核所需的文件化基础，包括来料控制、变更控制、追溯和出货检验。
5. 对外项目是否仅需 `日本本地准入`，还是同时需要 `国际安全表达`、`融资保险支持` 或 `公共项目/NITE` 路线。

## 2. 日本法人先承担什么责任

如果以日本法人做品牌、进口、销售或交付，至少要先把责任链写清楚：

1. 谁是 `日本品牌方`
2. 谁是 `日本进口主体`
3. 谁是 `日本销售主体`
4. 谁是 `中国制造主体 / 组装工厂`
5. 谁是 `电芯 / BMS / PCS / 逆变器 / 适配器` 的关键供应商
6. 谁是 `安装/施工/售后主体`
7. 谁持有认证、谁保管试验记录、谁应对召回

说明：

- METI 英文程序页明确写到：只有日本居住者或依日本《公司法》注册的公司，才具备进口业务主体资格。  
- 因此，“日本法人路线”并不减轻责任，而是将责任集中到日本本地实体。
- 在当前这条商业模式下，**日本法人是市场准入责任主体，中国工厂是制造实现主体**。
- 这意味着：
  - `PSE届出`、`JET对象/非对象判断`、日本市场标签与召回责任，原则上要由日本法人主导。
  - `工厂审查`、`Follow-Up`、`定期工厂调查`，实际落点往往会落到中国制造/组装工厂。
  - `IEC / UL / JET` 申证资料包必须同时覆盖日本品牌主体与中国制造主体，不能只写其中一方。

## 4. 便携式储能电源的最新分类结论

### 4.1 METI 当前最新可确认口径

按 `2026-06-25` 可确认的 METI 页面与 FAQ：

1. `モバイルバッテリーに関するFAQ`
   - Q.4 明确写明：可输出 `交流100V` 的所谓 `ポータブル電源`，因为蓄电池原理上输出为直流，所以 **不作为“蓄电池 / モバイルバッテリー”处理，属于非対象**。

2. `ポータブル電源の安全性要求事項（中間とりまとめ）`
   - METI 明确写明：**ポータブル電源目前是电气用品安全法的规制对象外**。
   - 同时，METI 又因为其存在火灾、触电等风险，单独发布了中间要求事项。

3. `調査概要`
   - METI 还特别提醒：`ポータブル電源` 这一叫法本身 **没有法令上的正式定义**。
   - 这意味着判断时不能只看营销名称，必须看 `主功能`、`输出形态`、`是否 AC 输出`、`是否可脱卸电池包`。

### 4.2 圆形 PSE 是否必要

#### 结论

- **对 AC 输出便携式储能电源整机：当前不要求圆形 PSE。**
- **对 DC 外付け电源 / モバイルバッテリー：圆形 PSE 仍然必要。**

#### 容易出错的边界

1. 如果产品主要功能仍是给手机、平板、PC 做外接 DC 供电，即使带简单附加功能，也可能仍按 `モバイルバッテリー` 处理。
2. 如果电池包可脱卸并单独流通，则被装上的 `リチウムイオン蓄電池` 仍可能作为对象处理。
3. 如果产品包含单独流通的 `直流電源装置` 等对象部件，部件级 PSE 义务要单独看。

#### 实务建议

- 对任何“长得像 portable power、但销售文案又强调给电子设备供电”的产品，先向 JET 申请一次 **对象/非对象文书判断**，不要靠销售文案自判。

## 5. 四类产品的认证路线图

## 5.1 A 类：AC 输出便携式储能电源

### 最小合规路线

1. **对象/非对象确认**
   - 先用规格书、方框图、接口表确认主功能是否为 AC 输出便携电源。
   - 必要时向 `JET` 申请电安法 `対象・非対象` 文书回答。

2. **整机安全路线**
   - 不走“强贴圆形 PSE”的错误路线。
   - 走 `JET S-JET認証（ポータブル電源）` 或等效第三方安全评估路线更合理。
   - 技术依据核心是 METI 公开的 `ポータブル電源の安全性要求事項（中間とりまとめ）`。

3. **供应链与出货资料**
   - `UN38.3`
   - `SDS/MSDS`
   - 海运/危特分类（若容量和运输形态触发）

4. **若带通信/远程控制**
   - 预评估 `JC-STAR` 是否会落到网关、App 控制器、EMS 子板上。

### 推荐增强项

- `S-JET` 对日本渠道、零售、电商和 B2B 采购沟通具有明显的降阻作用；是否构成硬性门槛，应以具体渠道规则和采购文件为准
- 若想做日本办公/公共设施/防灾采购，提前准备：
  - 待机自耗电
  - 长日历寿命
  - 充电器/适配器规格
  - 端口温升
  - AC 输出异常保护

### 主要风险点

- 把 `ポータブル電源` 当 `モバイルバッテリー` 强行贴圆形 PSE
- 只有 `UL/IEC`，却没有日本文书化的对象判断
- 只拿英文资料，不做日文版安全说明、警告标签、售后责任链

---

## 5.2 B 类：DC 外付け电源 / モバイルバッテリー

### 最小合规路线

1. 日本法人作为制造/进口主体完成电安法届出
2. 按 METI 口径完成技术基准适合确认
3. 做定格电压、外观等全数检查并保存记录
4. 整机标注：
   - `圆形 PSE`
   - 届出事業者名
   - 定格電圧
   - 定格容量

### 技术路径

- METI FAQ 明示：
  - 可按 `别表第9`
  - 或按 `别表第12` 采用 `J62133 / JIS C 8712:2015`
- JET 2026 年页面还提示：对モバイルバッテリ，JET 从安全性角度更推荐 `J62368-1 + J55032` 的组合路线。

### 结论

- 若产品本质上属于“手机/平板/PC 的外接 DC 电源”，则不应尝试绕开 `PSE` 路线。

---

## 5.3 C 类：低压并网型储能系统（户储 / 小型工商业）

### 该类产品的判断重点

该类产品最容易出现准入路径误判。  
其核心主线不是单纯以 `PSE` 为起点，而是：

1. **电池系统国际基线**
   - `IEC 62619`

2. **PCS / 电力电子安全**
   - `IEC 62477-1`

3. **EMC**
   - 工程上至少要有与系统/PCS 匹配的 `EMC` 试验表达

4. **日本低压储能系统本地认证**
   - `JIS C 4412`
   - `JET S-JET認証（蓄電システム）`

5. **并网 / 系统连系**
   - `JET 系統連系保護装置等認証`
   - 日本电网存在东日本 `50Hz`、西日本 `60Hz` 的双频差异，并网设备需按照系统连系技术要件证明其在频率、电压等异常条件下具备保护与切离能力

6. **网络安全**
   - 具 `Wi-Fi` 监控、远程控制、EMS 网关等 IP 通信能力的系统，建议在功能冻结阶段前置开展 `JC-STAR` 预评估
   - 对政府采购、关键基础设施和高安全要求采购场景，`JC-STAR` 的重要性会明显上升，但不宜在一般商业项目中一律写成“必备准入证”

7. **运输与交付**
   - `UN38.3`
   - `SDS/MSDS`
   - 进口物流资料

### 为什么 JIS C 4412 / S-JET 要前置

JET 2026 年蓄电池页面明确写到：

- `S-JET認証（蓄電システム）` 是基于 `JIS C 4412` 的低压蓄电系统产品认证。
- 它不仅检查 JIS 适合性，还加了电波障害防止等追加要求。
- 其要求中已包含和 `系統連系保護装置等認証` 同一的部分内容，因此有利于后续并网认证。

### 结论

- 如果是日本户储/小工商业储能，**真正的“必须路线”是 `JIS C 4412 + JET系統連系`，不是“先拿欧美证书再慢慢看日本”。**

## 5.4 D 类：大型定置储能系统（柜体 / 集装箱 / 公共基础设施）

### 最小合规路线

1. **电池安全**
   - `IEC 62619`

2. **PCS / 变换设备安全**
   - `IEC 62477-1`

3. **系统级国际安全表达**
   - `IEC 62933-5-2`

4. **日本本地系统级表达**
   - `JIS C 4441`
   - `JET JIS C 4441 プロパゲーション試験認証` 或 JET 的大型定置蓄电系统量产品认证

5. **并网 / PCS / 系统连系**
   - `JET 系統連系保護装置等認証`（适用时）

6. **火灾 / 热蔓延 / 融资资料**
   - `UL 9540A`（注意：这是测试方法，不是单独的市场准入证）
   - 大客户、保险、融资与消防审批场景常同时看 `UL 1973 / UL 9540 / UL 9540A`

7. **公共调达 / 重要基础设施项目**
   - 2026 年起应前置看 `NITE 公共調達・重要インフラ向け蓄電池システムの安全ガイドライン`
   - JET 已开始提供按该 NITE 指南出具适合证明书的服务
   - 对公共项目、地方政府和关键基础设施项目，应将该路线视作高门槛路线；是否构成强制要求，应以具体招标文件、补贴规则或采购制度为准

### 为什么 JIS C 4441 很关键

JET 2026 年页面明确写到：

- `JIS C 4441` 要求基于 `JIS C 8715-2` 的 `プロパゲーション試験`
- 该认证用于确认即使内部单电芯热暴走，也不会有火焰露出到系统外
- `JIS C 4441` 是日本提出的、与 `IEC 62933-5-2` 国际整合的标准

### 结论

- 日本大储路线不能只拿 `IEC 62933-5-2` 或 `UL 9540A` 对客户说“我们已经够了”。
- 更适合内部与外部沟通的表述是：
  - 国际说明语言：`IEC / UL`
  - 日本本地落地语言：`JIS C 4441 / JET / NITE`

1. `公开费用` 只写官网或官方费表可直接核验的数字。
2. `参考量级` 用于预算抓大头，不等于正式报价；对没有公开费表的项目，我只写公开市场案例或同类认证收费作参考，并明确标注“非官方”。

## 7. 建议执行顺序与实施重点

### 7.1 A 类：便携式储能电源

1. 先完成产品边界判定，明确属于 `AC 输出便携电源` 还是 `DC モバイルバッテリー`。
2. 以 `日本法人品牌型号` 为基础完成 `JET 对象/非对象` 判断。
3. 对 AC 输出便携式储能电源，原则上不以 `圆形 PSE` 为首证，而以 `S-JET（ポータブル電源）` 为日本本地安全表达。
4. 同步完成 `UN38.3`、`SDS/MSDS`、日文警告标签以及日本法人售后/召回责任链准备。

### 7.2 C 类：户储 / 小型工商业储能

1. 先完成 `IEC 62619 + IEC 62477-1 + EMC` 的国际基线准备。
2. 面向较强尽调客户、工程项目或海外资金方时，同步推进 `IEC / EN 62477-1` 第三方 CoC。
3. 日本本地准入层面，依次推进 `JIS C 4412 / S-JET` 与 `JET 系统连系`。
4. 对具备 IP 通信能力的系统，提前评估 `JC-STAR`。
5. 补齐日文规格书、施工条件、并网适配表等面向市场和施工的文件。
6. 采用中国工厂组装时，应同步准备工厂审核窗口、生产一致性文件和关键零部件 `BOM` 锁定表。

### 7.3 D 类：大储 / 柜体 / 集装箱

1. 先完成 `IEC 62619`、`IEC 62477-1`、`IEC 62933-5-2`、`UN38.3 / SDS` 等国际基线。
2. 对外项目销售、融资、保险或总包谈判阶段，优先形成 `IEC / EN 62477-1` 与 `IEC 62933-5-2` 第三方 CoC。
3. 日本本地准入层面，推进 `JIS C 4441`、`JET` 及适用场景下的 `系統連系`。
4. 面向公共调达、基础设施、公共项目时，进一步评估 `NITE ガイドライン` 路线。
5. 面向海外融资、保险或北美市场时，再评估 `UL 1973 / UL 9540 / UL 9540A` 的必要性。

### 7.4 当前模式下的主要实施风险

1. 日本品牌名、型号名、说明书名已经确定，但中国工厂版本号、BOM 版本号没有同步冻结。
2. 认证申请写的是日本法人，却没有把中国制造工厂列入后续工厂调查和一致性管理清单。
3. 日本销售资料写的是品牌规格，认证资料交的是 ODM/OEM 旧型号，导致标签、铭牌、报告不一致。
4. 只准备了整机资料，没有同步准备电芯、BMS、PCS、适配器等关键件的上游证据包。

## 8. 汇报建议

### 8.1 必须写进立项输入的认证项

#### 便携 AC 输出电源

- `JET 对象/非对象判断`
- `S-JET（ポータブル電源）`
- `UN38.3`
- `SDS/MSDS`

#### 户储 / 小工商业

- `IEC 62619`
- `IEC 62477-1`
- `IEC / EN 62477-1` 第三方 CoC（面向项目客户或较强尽调时）
- `EMC`
- `JIS C 4412 / S-JET`
- `JET 系统连系`
- `JC-STAR` 预分析
- `UN38.3 / SDS`

#### 大储 / 柜体 / 集装箱

- `IEC 62619`
- `IEC 62477-1`
- `IEC 62933-5-2`
- `IEC / EN 62477-1` 第三方 CoC
- `IEC 62933-5-2` 第三方 CoC
- `JIS C 4441 / JET`
- `JET 系统连系`
- `UL 1973 / UL 9540 / UL 9540A`（融资/国际客户增强）
- `UN38.3 / SDS / 运输资料`
- `NITE ガイドライン`（公共项目型）

### 8.2 最容易踩的坑

1. 把所有日本储能产品都先问 `PSE`
2. 把 `UL 9540A` 当成“完整认证”
3. 只拿国际证书，不做 `JET / JIS / 并网 / 日文资料`
4. 不先做对象/非对象判断，就开始量产或贴标
5. 预算里没有留出工厂调查、Follow-Up、定期审查和资料翻译成本

### 8.3 建议执行策略

建议先以一条样板产品线完成闭环验证，再扩展到其他场景。当前更稳妥的推进顺序为：

1. 明确一个产品形态和一个目标市场入口。
2. 完成一份 `对象/非对象` 判断。
3. 跑通一条日本本地认证路线。
4. 视客户与项目需要补齐国际认证包。
5. 形成一套可对外复用的日文资料包。

不建议在同一阶段同时并行推进 `便携电源 + 户储 + 工商业柜 + 大储集装箱` 四条线。

## 9. 名词解释：日本市场常用储能专业术语

说明：

- 本节只解释本文档中高频出现、且容易混淆的术语。
- 若术语同时具有法令定义和行业习惯用法，本文优先按“日本市场实务判断”解释。

| 术语 | 常见原文 | 本文中的实际含义 |
|---|---|---|
| `PCS` | `Power Conditioning System` / `パワーコンディショナ` | 储能系统中的功率变换核心设备，负责电池直流与交流电网/负载之间的变换，通常也承担充放电控制、并网接口和异常切离逻辑。在日本户储、小工商储和大储项目中，`PCS` 往往是并网认证和系统连系判断的核心设备。 |
| `BMS` | `Battery Management System` | 电池管理系统，负责电芯电压、电流、温度监控、均衡和保护。便携储能更偏向“电池包保护与状态管理”，户储/工商业储能则更强调与 `PCS`、`EMS` 的联动。 |
| `EMS` | `Energy Management System` | 能源管理系统，负责更上层的调度、监控、数据采集和远程控制。若产品带 `Wi-Fi`、云端监控、远程控制或网关功能，`EMS` 往往也是 `JC-STAR` 预评估的重点边界。 |
| `PSE` | `電気用品安全法` 标识 | 日本《电气用品安全法》下的合规标识。本文特别强调：不是所有储能产品都以整机 `PSE` 为主线，应先区分产品是否属于受规制对象。 |
| `圆形 PSE` | `丸形PSE` | 常见于特定 `电池/电源类` 对象产品。本文语境下，主要对应 `モバイルバッテリー` 等 DC 外接供电产品，不直接等同于 AC 输出便携储能整机。 |
| `METI` | `経済産業省` | 日本经济产业省。本文中，`METI` 是电安法、便携式储能分类和中间要求事项的主要官方口径来源。 |
| `JET` | `一般財団法人電気安全環境研究所` | 日本电气安全环境研究所。本文中既是 `対象・非対象` 判断、`S-JET`、`系統連系` 等关键路径的官方机构，也是日本储能准入判断的高频接口。 |
| `S-JET` | `S-JET認証` | 日本的第三方产品安全认证。对 AC 输出便携式储能电源不是法定强制，但在渠道沟通、采购准入和风险控制中具有很强的现实价值。 |
| `JIS` | `Japanese Industrial Standards` | 日本工业标准。本文中常见于 `JIS C 4412`、`JIS C 4441` 等储能系统级标准，是日本本地认证和项目交付的重要技术语言。 |
| `系統連系` | `系统连系` / `grid interconnection` | 指储能系统与日本电网并接运行时所需满足的技术要求和认证路径。只要产品涉及并网、逆潮流、并网切离等功能，通常都要单独看 `系統連系`。 |
| `JC-STAR` | `Labeling Scheme based on Japan Cyber-Security Technical Assessment Requirements` | 日本 IoT 产品网络安全标签制度。本文中主要针对带通信、远程控制、网关或云平台能力的储能系统，不是所有储能产品都自动适用。 |
| `NITE` | `製品評価技術基盤機構` | 日本产品评价技术基盘机构。本文主要关注其面向公共调达和重要基础设施项目的储能安全指南。 |
| `EMC` | `Electromagnetic Compatibility` | 电磁兼容。本文中是国际安全与系统级交付的基础项，尤其对 `PCS`、逆变器、并网设备不能省略。 |
| `UN38.3` | `UN Manual of Tests and Criteria, Part III, Sub-section 38.3` | 锂电池运输测试要求。本文中主要用于便携储能、电池包和系统出货环节，不替代市场准入认证。 |
| `SDS / MSDS` | `Safety Data Sheet` | 安全数据表。本文中主要用于运输、报关、仓储和客户交付资料，不等同于产品认证。 |
| `SAT` | `Site Acceptance Test` | 现场验收测试。本文中特别用于大储项目语境，表示即使已有系统级 CoC，落地站点仍需按项目条件单独验收。 |
| `OEM / ODM` | `Original Equipment Manufacturer / Original Design Manufacturer` | 分别指代工生产和代设计代工。本文中最重要的不是概念本身，而是品牌型号、认证型号和工厂内部型号必须一一映射，不能混用。 |
| `BOM` | `Bill of Materials` | 物料清单。本文中既是成本测算基础，也是认证一致性管理和变更控制的基础文件。 |
| `ポータブル電源` | 日本市场对 AC 输出便携储能电源的常见叫法 | 本文语境下，重点指可输出 `AC 100V` 的便携式储能电源，不应简单等同于受电安法规制的 `モバイルバッテリー`。 |
| `モバイルバッテリー` | 日本市场对移动电源/外接 DC 供电电池的常见叫法 | 本文语境下，主要指给手机、平板、PC 等提供外接 DC 电源的产品，通常仍需按 `圆形 PSE` 路线判断。 |
