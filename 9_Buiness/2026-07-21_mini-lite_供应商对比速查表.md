# mini-lite 供应商对比速查表（会前快读版）

## 1. 使用说明

这份表不是为了做行业全景，而是为了支撑 `mini-lite / STPP-64A-W` 谈判前快速判断：

- 谁适合当前快落地；
- 谁适合做功率升级；
- 谁更像备胎或替代方案；
- 哪些宣传点已经有公开信息支撑，哪些仍然只是附件单方说法。

信息等级说明：

- `A`：有官方/公开页面可交叉核对
- `B`：主要来自附件内容，需会谈中拿证据
- `C`：公开信息不足，必须现场确认

## 3. 站在 mini-lite 角度的判断

### 第一优先级：智融

原因：

- 现有资料已经是智融体系；
- 主控与协议层是当前项目的关键稳定点；
- 最容易走“最小改动、最快打样”路线。

### 第二优先级：红与蓝 / 镓未来（二选一做功率层备选）

原因：

- 如果后续要做更薄、更低温、更高端外观，GaN 路线值得评估；
- 但在主控基线未完全冻结前，不适合让功率层方案反客为主。

### 第三优先级：英诺赛科

原因：

- 更适合作为供货稳定性、可靠性和技术上限的对照组；
- 也是对谈判对象施压时的“强参考”。

### 第四优先级：镓创晶合 / 华燊泰

原因：

- 可以作为补充比价与替代资源；
- 但从当前 mini-lite 项目速度与把控力来看，不是首选。

## 4. 谈判时最该盯住的五个维度

### 4.1 核心器件是不是当前推荐量产料

不要只问“有没有这个料”，要问：

- 是否仍为主推料号；
- 生命周期还有多久；
- 是否存在替代料切换计划；
- 交期、起订量、近期缺货风险如何。

### 4.2 方案是不是能最小改动映射到 mini-lite

重点不是“你们最强方案是什么”，而是：

- 现有 `45W + 18W + 65W` 结构能否直接映射；
- 现有板子要改哪里；
- 是否需要换主控 / 换外围 / 换电感 / 换功率管；
- 这样改后是否会影响外壳、热设计和认证节奏。

### 4.3 价格不能只看芯片单价

会谈时要把总成本拆开问：

1. 芯片/器件单价
2. 是否绑定指定物料
3. 改板费
4. 调试费
5. 认证整改支持费
6. 小批量样机支持费用

否则容易出现：

- 料单便宜，但后续服务收费高；
- 公版免费，但必须锁指定器件；
- 前期低价，后期切料困难。

### 4.4 FAE 是不是能真正“到场闭环”

要区分三种支持：

1. 只发资料
2. 远程答疑
3. 跟样机、跟实验室、跟量产异常闭环

`mini-lite` 要的是第 3 种。

### 4.5 认证支持不能只听“可以做”

要落到：

- 做过哪些日本向项目；
- 做过哪些失效整改；
- 提供哪些资料；
- 是否有合作实验室；
- 出问题后谁负责跟。

## 5. 建议的采购/合作策略

### 建议策略

1. 主线先锁 `智融`
2. 功率层平行评估 `红与蓝` 与 `镓未来`
3. 用 `英诺赛科` 做技术与供货上限参考
4. 用 `华燊泰/镓创晶合` 做价格与备份触点

### 不建议策略

- 直接把整机路线押给某一家方案商；
- 在没有样机数据前就承诺独家；
- 因为“免费公版”忽略锁料和后续工程费。

## 6. 会前一句话提醒

这次不是去谈“谁技术概念最大”，而是去判断：

`谁能最低风险、最短时间、最清晰边界地把 mini-lite 做出来，并支撑日本市场导入。`
