# Day03：低压储能核心技术卡片

## 1. 今日学习目标

今天的目标不是继续讲法规，而是建立低压储能系统的最小技术知识树，能够讲清：

1. LFP、BMS、SOC/SOH、PCS、EMS 各自是什么；
2. 它们各自负责什么、不负责什么；
3. 它们如何组成一套低压储能系统；
4. 它们与 Day1 的场景判断、Day2 的合规初筛分别是什么关系。

## 2. 核心技术对象总表

| 对象 | 它是什么 | 它直接负责什么 | 不负责什么 | 对业务的意义 |
|---|---|---|---|---|
| LFP | 一种锂电池化学体系 | 提供储能单元本体 | 不负责系统调度和并网控制 | 决定安全性、寿命、能量密度和成本基础 |
| BMS | 电池管理系统 | 监控电压、电流、温度，做保护和均衡 | 不直接决定什么时候充放电给电网 | 决定系统能不能安全用、稳定用 |
| SOC | 电池当前剩余电量状态 | 回答“还剩多少电” | 不回答电池老化程度 | 决定用户显示和调度判断 |
| SOH | 电池健康状态 | 回答“电池还健康不健康” | 不等于当前剩余电量 | 决定寿命、售后和残值判断 |
| PCS | 功率变换系统 | 负责 AC/DC 双向变换，并网或离网输出 | 不负责电芯级安全管理 | 决定系统怎么和电网、负载交换能量 |
| EMS | 能量管理系统 | 决定何时充、何时放、如何调度 | 不直接代替 PCS 做功率变换 | 决定系统能不能实现经济性和策略控制 |

## 3. 技术卡片 1：LFP（Lithium Iron Phosphate, LiFePO₄）

### 1. 一句话定义
LFP 是磷酸铁锂电池化学体系，是储能系统里最底层的能量载体之一。

### 2. 它在系统里负责什么
它负责“把电存起来”，决定系统的安全基础、寿命基础、容量基础和部分成本基础。

### 3. 它不负责什么
它不负责系统调度，不负责并网控制，也不直接决定用户界面显示。

### 4. 日本低压储能场景里为什么重要
日本市场对安全、长期待机、室内使用、长期维护稳定性更敏感，因此电芯体系不是只看能量密度，还要看热稳定性和寿命。

### 5. 业务上最容易讲错的点
不能把“用了 LFP”直接等同于“整个系统就一定安全”；系统安全还取决于 BMS、PCS、结构、散热和保护策略。

## 4. 技术卡片 2：BMS

### 1. 一句话定义
BMS 是电池管理系统，是电池包的安全总管和状态监控层。

### 2. 它在系统里负责什么
它负责监控单体和系统级的电压、电流、温度，执行过充、过放、过流、过温等保护，并在需要时做电芯均衡和状态上报。

### 3. 它不负责什么
它不直接把直流变成交流，也不负责决定整个系统什么时候为了电价策略去充放电。

### 4. 日本低压储能场景里为什么重要
无论是公寓防灾、一户建备电，还是低压并网系统，日本市场都高度重视失效风险和安全解释责任，BMS 是最核心的安全责任接口之一。

### 5. 业务上最容易讲错的点
不能把 BMS 说成“一个显示电量的小板子”；它本质上是电池安全和状态管理系统。

## 5. 技术卡片 3：SOC / SOH

### 1. 一句话定义
SOC 是荷电状态，回答“还剩多少电”；SOH 是健康状态，回答“电池还剩多少可用健康度”。

### 2. 它在系统里负责什么
SOC 用于显示剩余电量、续航时间和调度判断；SOH 用于判断电池老化、维护计划和寿命状态。

### 3. 它不负责什么
SOC 和 SOH 都不是功率变换器，也不是保护执行器；它们是状态量，不是硬件执行层。

### 4. 日本低压储能场景里为什么重要
日本用户买的不只是“电池”，而是“停电时的确定性”。如果 SOC 不准，用户会误判可用时间；如果 SOH 不清，长期维护和售后就会失真。

### 5. 业务上最容易讲错的点
最常见错误是把 SOC 当成 SOH。电量剩 80% 不代表电池健康还有 80%。

## 6. 技术卡片 4：PCS

### 1. 一句话定义
PCS 是功率变换系统，负责在电池直流电和交流电网/交流负载之间做双向能量转换。

### 2. 它在系统里负责什么
它负责充电、放电、并网输出、离网供电、功率控制、电能变换质量等核心功率路径。

### 3. 它不负责什么
它不负责单体电芯级别的安全监控，也不单独定义整套系统的运行策略。

### 4. 日本低压储能场景里为什么重要
只要涉及 AC 负载、并网、离网切换或电网适配，PCS 就是系统成立与否的关键设备。

### 5. 业务上最容易讲错的点
不能把 PCS 和逆变器简单完全等同。逆变只是它的一部分语境，PCS 更强调双向和系统级功率控制。

## 7. 技术卡片 5：EMS

### 1. 一句话定义
EMS 是能量管理系统，负责定义和执行储能系统的运行策略。

### 2. 它在系统里负责什么
它负责根据电价、负载、并网条件、储能状态和运行目标，决定何时充电、何时放电、何时待机、何时限功率运行。

### 3. 它不负责什么
它不直接替代 PCS 做功率变换，也不替代 BMS 做单体安全保护。

### 4. 日本低压储能场景里为什么重要
在并网型、小型工商型或多场景复合应用中，EMS 直接决定系统是“只是能用”，还是“能真正产生经济价值”。

### 5. 业务上最容易讲错的点
不能把 EMS 说成“就是 App”。App 只是界面，EMS 是策略逻辑本体。

## 8. 系统关系图（文字版）
一个低压储能系统的最小关系可以理解为：

LFP 电芯
-> 组成电池模组 / 电池包
-> 由 BMS 监控与保护
-> 通过 PCS 与交流电网或交流负载交换能量
-> 由 EMS 决定系统何时充电、何时放电、如何运行
-> 通过 SOC / SOH 等状态量向用户和系统提供判断依据
