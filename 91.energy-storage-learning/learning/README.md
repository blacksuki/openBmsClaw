# Learning

本目录保存学习过程材料。

学习材料允许逐步完善，但必须做到：

1. 标题清楚。
2. 日期清楚。
3. 结论、证据、待验证项分开。
4. 能回链到对应的 roadmap day。

## 子目录

| 目录 | 用途 |
|---|---|
| [notes/](notes/README.md) | 每日学习笔记、概念卡片、访谈记录 |
| [diagrams/](diagrams/README.md) | 市场图、供应链图、系统结构图 |
| [summaries/](summaries/README.md) | 阶段总结、主题摘要、一页纸 |
| [homework/](homework/README.md) | 作业、测验、导师批改稿 |

## 文件命名建议

```text
Day01_japan-low-voltage-storage-scenarios.md
Day02_japan-compliance-route.md
Day03_core-technology-cards.md
Week01_review.md
```

保留 `DayXX` 前缀，便于和 [../roadmap/](../roadmap/README.md) 对齐。
