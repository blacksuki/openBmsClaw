# Homework

这里保存导师布置的作业、测验和批改记录。

## 作业规则

1. 每份作业必须有明确问题。
2. 输出必须能被 reviewer 审查。
3. 不确定内容要写入 `待验证项`。
4. 被批改后，不覆盖原稿；新增 `reviewed` 或 `revision` 文件。

## 推荐命名

```text
Day01_homework_scenario-map.md
Day01_homework_scenario-map_reviewed.md
Day01_homework_scenario-map_revision.md
```
