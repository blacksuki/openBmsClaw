# Roadmap

本目录保存学习路线和阶段目标。

当前主线从 [week01.md](week01.md) 和 [week02.md](week02.md) 开始，来源于原始 [2周储能行业专家速成计划](../../90_documents/learning/2周储能行业专家速成计划.md)。

## 文件说明

| 文件 | 用途 |
|---|---|
| [roadmap.md](roadmap.md) | 总体路线和能力层级 |
| [week01.md](week01.md) | Week 1：认知解构与供应链摸底 |
| [week02.md](week02.md) | Week 2：商业重构与市场进入 |

## 使用方式

1. 每周开始前读对应 `weekXX.md`。
2. 每天只聚焦一个输出物。
3. 每天结束后把产出放入 [../learning/](../learning/README.md) 或 [../research/](../research/README.md)。
4. 周末用 [../reviews/weekly-review-template.md](../reviews/weekly-review-template.md) 复盘。
