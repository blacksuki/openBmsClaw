# Agents

本目录保存长期稳定的 AI 角色 Prompt。

Agent 只回答“怎么教、怎么审、怎么研究、怎么翻译”，不直接保存 Week 1、Week 2 这样的学习计划。学习路线放在 [../roadmap/](../roadmap/README.md)。

## Agent 列表

| 文件 | 角色 | 使用场景 |
|---|---|---|
| [mentor.md](mentor.md) | 长期导师 | 日常学习、讲解、追问、作业布置 |
| [reviewer.md](reviewer.md) | 学习成果审稿人 | 审阅每日输出、发现逻辑漏洞 |
| [researcher.md](researcher.md) | 行业研究员 | 收集和整理市场、供应链、竞品、法规证据 |
| [translator.md](translator.md) | 日文资料分析员 | 解析日本官网、标准说明、产品页和行业资料 |

## 使用方式

1. 学习新主题时，先用 `mentor.md`。
2. 完成作业后，用 `reviewer.md` 做结构和证据审查。
3. 需要补充事实时，用 `researcher.md`。
4. 遇到日文原文、法规、认证机构资料时，用 `translator.md`。

不要让一个 Agent 同时扮演所有角色。角色拆开后，输出更稳定，也更便于长期维护。
