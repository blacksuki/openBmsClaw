# Research

本目录保存行业研究材料。

研究材料必须区分：

- 已验证事实
- 高可信行业共识
- 假设
- 推论
- 待验证项

## 子目录

| 目录 | 用途 |
|---|---|
| [japan-market/](japan-market/README.md) | 日本市场、用户场景、渠道、需求 |
| [suppliers/](suppliers/README.md) | 中国供应链、供应商、询盘、规格书 |
| [competitors/](competitors/README.md) | 标杆企业、竞品、产品矩阵 |
| [regulations/](regulations/README.md) | PSE、JET、JIS、JC-STAR、NITE、并网 |

## 研究原则

1. 法规和认证优先引用官方或认证机构来源。
2. 竞品参数优先引用官网、说明书、认证文件。
3. 用户需求优先引用一手用户反馈和真实商品评论。
4. 供应链结论优先引用询盘、规格书、报价单和工厂资料。
5. 没有证据时不要写成结论。
