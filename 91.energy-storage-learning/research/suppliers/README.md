# Suppliers Research

这里保存中国供应链和供应商研究。

## 供应链分层

| 层级 | 角色 | 关键问题 |
|---|---|---|
| 电芯 | LFP / NCM cell supplier | 安全、寿命、认证、供货稳定性 |
| BMS | Battery Management System | 采样、保护、均衡、通信、日志 |
| PCS / 逆变器 | Power conversion | 输出、并网、自立、效率、认证 |
| EMS | Energy Management System | 策略、远程监控、数据、安全 |
| PACK | 结构、电气、热管理集成 | 一致性、工艺、测试、追溯 |
| 认证资料 | 测试、证书、标签、说明书 | 型号一致性、变更控制、日文资料 |
