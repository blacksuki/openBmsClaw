## 标杆企业优先级

| 企业 | 重点学习点 |
|---|---|
| Sungrow | PCS + ESS + 日本本地认证/市场动作 |
| Huawei Digital Power | 日本向户储资料、光储融合、证书公开方式 |
| CATL | 电芯到系统平台、认证矩阵、系统产品化 |
| HiTHIUM | 储能专用公司、日本办公室、系统安全表达 |
| Ganfeng | 电芯、系统、认证包、资料包组织方式 |

## 竞品矩阵字段

| 字段 | 说明 |
|---|---|
| 场景 | 便携、户储、小工商、大储、办公移动电源 |
| 容量/功率 | Wh/kWh、W/kW、浪涌能力 |
| 输出/输入 | AC、DC、USB-C PD、太阳能、行车充、并网 |
| 安全 | 电芯体系、BMS、热管理、消防、认证 |
| 日本本土化 | 日文资料、标签、渠道、售后、认证 |
| 价格 | 官方价、渠道价、活动价 |
| 弱点 | 重量、噪音、资料缺口、认证缺口、售后风险 |

## 输出要求

不要只做参数表。每份竞品分析必须回答：

1. 它服务哪个场景？
2. 它解决了什么日本本土问题？
3. 它的产品定义强在哪里？
4. 它暴露了什么机会缺口？
