## 场景 1：首都圈公寓防灾

### 物理边界

- 东京共同住宅对“避难上有効なバルコニー”的行政解释要求：
  - 有效奥行 `>=75cm`
  - 宽 `>=150cm`
  - 邻户隔板处应确保约 `60cm` 的通过宽度  
  这意味着：公寓阳台上的储能设备不能占用逃生通路，尤其不能堵住隔板和避难动线。

- 东京 2026 年《国の首都直下地震における被害想定の分析と首都東京の強靭化に向けた都の見解》给出的电力分析里，`被災1週間後` 的停电率仍按 `3%` 处理；文件同时说明这并不代表完全恢复，而是包含计划停电/节电前提。  

### 推论

- 公寓防灾的第一约束不是“买多大”，而是 `设备不能侵占逃生线`。
- 第二约束不是“给手机充电”，而是 `电梯停摆后，冰箱/风扇/呼吸辅助类设备还能不能撑到恢复或转移`。
- `3%停电率` 的含义不是“问题很小”，而是 `日本防灾电源更像10年耐用品`：平时长期待机，出事时一次就要能救命。
- 因此，公寓场景不应默认推大箱体；先问 `阳台是否能留出60cm逃生线`、`室内是否有固定位置`、`要保哪些关键负载`、`电梯停摆后谁来搬运`。

- 一户建场景的核心不是“有没有电”，而是 `能不能自己撑过72小时到1周`。

- 因此一户建方案必须先分层：
  1. `8小时级` 关键负载底线
  2. `72小时~1周` 生活维持层
  3. 再谈整屋舒适层

## 四场景共通结论

1. 日本用户不是泛泛“怕断电”，而是在 `各自物理边界` 内怕断电。
2. 东京/大阪真实场景的第一约束，常常不是 `Wh`，而是：
   - 公寓：逃生线
   - 一户建：多日连续自持
   - 车中泊：补能/固定/防水
   - 小型商用：输出匹配与运维
3. 东京 `被災1週間後仍按3%停电率处理` 这件事，进一步说明：日本防灾电源首先是 `长日历寿命、低自放电的耐用品`，其次才是高频循环设备。
4. 因此低压储能四大场景不能共用一个销售话术，也不能共用同一套产品定义。

## 提交前自检答案

### 1. 东京都为何禁止阳台放置过深/过大的储能设备

不是单纯“嫌难看”，而是因为共同住宅阳台在法规和行政解释里承担 `避难通路` 角色。  
东京对“避难上有効なバルコニー”的解释要求有效奥行 `>=75cm`、宽 `>=150cm`，并要求隔板处留出约 `60cm` 的通过宽度。  
所以，问题本质不是“50cm 这个数字本身”，而是：一旦设备深度、室外机、杂物叠加后吃掉了 `60cm` 逃生线，就会构成避难障碍。  
建築基準法施行令第127条属于“敷地内の避難上及び消火上必要な通路等”的上位框架；在共同住宅阳台上，东京的建筑安全条例和运用基准把这件事落到了具体尺寸。

### 2. 日本“防灾”与“露营”场景对循环寿命的要求差异是多少

按“日本市场平均使用模型”做保守换算，可以给出一个**可复算**的差值：

1. `防灾待机模型`
   - Jackery 官方保管建议：`3个月检查/补电1次`
   - 若按 `10年持有`
   - 取最保守上限，按 `1次点检 = 1次等效满循环`
   - 则约为 `4次/年 × 10年 = 40次`

2. `露营平均模型`
   - 日本オートキャンプ協会《オートキャンプ白書2025》：年均 `5.0回`、`6.7泊`
   - 若按 `10年持有`
   - 低位模型：`1次出行 = 1次等效满循环`，约 `50次`
   - 高位模型：`1泊 = 1次等效满循环`，约 `67次`

3. `差值`
   - 露营低位模型 - 防灾待机模型 = `50 - 40 = 10次`
   - 露营高位模型 - 防灾待机模型 = `67 - 40 = 27次`

结论：  
按日本**平均露营频次**与**防灾待机点检频次**计算，二者的循环寿命差异并不大，约 `10-27次 / 10年`。  
真正拉开差距的，不是“防灾 vs 露营”这两个标签本身，而是用户是否进入 `高频车中泊/周末长期外出/日常自发自用` 模式。  
所以日本市场里，四场景的第一分层不该先按“循环寿命”讲，而要先按 `空间约束`、`关键负载`、`连续时长`、`补能条件` 讲。

### 3. 若用户说“要轻便”，必须追问的 3 个问题

1. 你说的“轻便”是指 `单手可搬`、`能进车内`，还是 `能放进公寓固定位置`？
2. 你要保的负载是什么：`手机/风扇`，还是 `冰箱/医疗/制冰/空调`？
3. 你要撑多久：`4小时`、`1晚`、`2泊3日`，还是 `72小时~1周`？

## 补充场景：无固定工位办公电源（用可移动电源给电脑供电）

### 场景定义

- 这不是“停电应急电源”，而是把 `插座` 变成 `可移动基础设施`。  
- 典型场景不是家庭，而是：
  - `フリーアドレス`（无固定工位）
  - `ABW`（按活动选座办公）
  - `ラウンジ / カフェスペース`
  - `临时扩容会议区`
  - `共享办公 / コワーキング`

### 日本网站 / 案例里已经出现的明确信号

1. `日本办公环境确实在走向自由工位，而不是假设场景`
   - GOOD PLACE 2026 年对 `441名` 在自由工位办公室工作的日本公司职员调查显示：
     - 仅 `44.2%` 认为“完全自由工位”最舒适
     - `57.6%` 的公司仍是“无规则自由选座”
     - 用 `App/系统预约座位` 的满意度是 `77.2%`
     - “无规则自由选座”满意度只有 `57.1%`
     - 最大不满项是“安静作业席不足” `29.7%`
   - 这说明日本办公室不是单纯“把固定席拆掉”就结束，配套基础设施必须跟上。

2. `办公电源已经被当成单独的空间规划问题`
   - Panasonic 的 ABW 办公专栏写得很直接：
     - 一般办公室常按 `10-12㎡` 配 `1个插座`
     - 单个工位通常需要 `3-5口` 电源
     - ABW 办公里约有 `10-20%` 的区域属于“很难优雅布线”的区域
     - 若 `50人` 办公、自由区利用率按 `10%`、再加备用电池，示例推导是 `16台` 电池
   - 这说明“移动电源办公”在日本不是玩具，而是正式的电源补完方案。

3. `日本企业已经在用，不只是家具商想象`
   - Panasonic 的 `りそな銀行 Resona Garage` 事例里：
     - 首批采用 `e-block 本体7台 + スタンド7台`
     - 后续又加 `e-blockデスク 4台`
   - 目的不是防灾，而是：
     - 员工增加后，避免再做电源增设工事
     - 会前临时聚集到大屏前时，补足电脑电源
     - 继续保持可移动桌位与自由布局

4. `产品形态已经分化成“个人随身型”与“共享场景型”`
   - `个人随身型`
     - ITOKI `hako`：约 `115.2Wh`，本体约 `990g`，`USB-C×2 + USB-A×1`，总输出 `100W`
     - KOKUYO `Energy bottle`：`207Wh`，`USB-C 65W + 18W` 两口，500ml 水瓶级体积，可放常见杯架
     - Okamura `OC`：约 `240Wh`，约 `1.9kg`，`USB-C×3 + USB-A×1`，官方按 `50Wh` 笔记本估算约 `3.5回分`
   - `共享场景型`
     - KOKUYO `Energyシリーズ`：搭载 `540Wh` 充电式电源，`4口` 插座，明确用于无固定电源的 `ラウンジ / カフェ空間`
     - Panasonic `e-block`：`304Wh` 电池本体 + 桌面/立式充放电器，支持给 `PC / モニター / スマホ` 等同时供电

### 这个产品场景的本质

- 本质不是“把大储能搬进办公室”，而是补 `固定插座不好布、但人又真的要在这里工作` 的那一段空白。
- 直接替代对象不是 `UPS`，也不是 `家用户外电源`，而是：
  - 地插
  - 墙插
  - 临时排插
  - 为了改布局反复做的电源工事

### 日本办公场景下，这类产品最重要的特征

1. `先看输出，不先看容量`
   - 办公主负载是 `笔记本 + 手机 + 平板 + 小显示器`
   - 所以第一指标往往是 `USB-C PD 65W级` 是否稳定，而不是先比大 Wh

2. `必须能融入自由工位动线`
   - 日本主流产品明显在压：
     - 重量
     - 单手携带
     - 桌面占用
     - 外观融入家具
   - 这说明办公室里用户讨厌的是“像施工现场一样拖线”，不是单纯缺电

3. `必须有运维方案，不是买来就完`
   - 预约座位已经能把满意度从 `57.1%` 拉到 `77.2%`
   - 同理，移动电源若没有：
     - 统一充电点
     - 余量显示
     - 借还规则
     - 备用池数量
   - 很快就会从“灵活”变成“找电池”

4. `它更适合“补电区”，不适合“全办公室去插座化”`
   - Panasonic 自己的 ABW 专栏已经暗示：真正难布线的，多是 `10-20%` 的开放区、咖啡区、会议临时区
   - 所以更合理的产品定位是：
     - `固定工位/核心執務区` 继续保留正式供电
     - `开放协作区/短时驻留区` 用移动电源补足

5. `安全与认证门槛高于普通消费电源`
   - 办公室是高频、多人、室内、长期使用
   - Panasonic 明确把 `PSE` 和 `Sマーク` 放在产品特长里讲
   - 这说明日本 B2B 办公客户买的不只是“能充电”，而是 `能不能放心在办公室规模化运维`

### 对 openBattery 视角的产品启发

- 这个补充场景对应的不是“家庭备灾电源”，而是 `办公室里的电源节点产品`。
- 如果以后看日本办公市场，这条线的产品定义应优先满足：
  1. `USB-C PD 65W级` 笔记本供电
  2. `1-2kg级` 单手携带，或桌面友好体积
  3. `可视化 SOC`（剩余电量一眼可见）
  4. `统一充电底座 / 充电柜`
  5. `外观融入办公家具`
  6. `PSE / 室内高频运维安全`
- 反过来说，如果产品太重、只强调 AC 大功率、外观像户外电源、没有充电底座，那它更像 `露营电源`，不太像日本办公室电源。

### comment

- 这个场景与前四大场景不同，它是 `高频日常使用型`，不是 `低频灾害待机型`。  
- 因此这里比“10年放着不用”更重要的，是 `日常运维效率`、`桌面友好性`、`多设备兼容性`、`规模化管理能力`。

### 3. 这张场景图说明了什么

1. 日本办公室里的移动电源，首先不是“灾害产品”，而是 `空间基础设施补丁`。
2. 最常见的缺口不是“整个办公室没电”，而是：
   - 开放区不好布线
   - 自由工位人比插座更会移动
   - 会议/活动区用途变化快
3. 所以日本办公室电源产品的直接竞争对手，不只是别家电池，更是：
   - 地插
   - 墙插
   - 临时排插
   - 电源增设工事

### 4. 日本办公移动电源的产品分层

| 产品层 | 日本已出现的典型形态 | 典型规格信号 | 更像什么产品 |
|---|---|---|---|
| `个人工位型` | ITOKI `hako`、KOKUYO `Energy bottle` | `100-200Wh`、`1kg` 级、`USB-C 65W` 左右 | “可随身带着走的办公室插座” |
| `共享协作型` | Okamura `OC`、Panasonic `e-block`、KOKUYO `Energyシリーズ` | `200-540Wh`、多口输出、可带 PC/显示器/多人设备 | “补电节点 / 可移动插座岛” |
| `Phase Free 办公型` | e-block 在银行、办公点、庁舎的方案 | 平时办公、停电转 BCP、强调认证与运维 | “办公室里的小型储能节点” |

### 5. 对 openBattery 的直接启发

- 如果未来切入日本办公设施市场，不要先用“户外电源”思路定义产品。
- 更合理的切入问法是：
  1. 这是 `个人随身插座`，还是 `共享补电节点`？
  2. 目标是替代 `固定插座`，还是替代 `临时排插和布线工事`？
  3. 卖点是 `桌面友好 + 运维效率`，还是 `停电时顺带能救场`？
- 从日本现有信息看，办公室场景最重的不是“极限大容量”，而是：
  - `输出匹配`
  - `好搬`
  - `好看`
  - `好管`
  - `能规模化复制`

### comment

- 若继续深挖这个方向，下一步最值得单独做的是：`日本办公室移动电源规格对照表`。  
- 重点字段应包括：`Wh`、`USB-C PD`、`AC 口数量`、`重量`、`是否有底座/充电柜`、`是否强调 PSE / Sマーク / フェーズフリー`。
