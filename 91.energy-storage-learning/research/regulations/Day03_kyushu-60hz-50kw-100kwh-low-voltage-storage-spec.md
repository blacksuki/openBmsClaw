# Day 3 作业：九州 60Hz 电网 50kW/100kWh 低压储能系统技术规格书关键条目

## 1. 适用边界

说明：

- 本文按 `日本九州地区（60Hz）` 的 `低压并网型蓄电系统` 重写关键规格条目。
- `50kW / 100kWh` 在日本实务上已明显超出普通独栋住宅常见户储容量，更接近 `低压小型工商业 / 公寓共用部位 / 小楼宇备电` 场景。
- 因此，本文将其定义为 `低圧蓄電システム`，并按 `分离形 PCS + 蓄电池系统` 的架构书写。

---

## 2. 技术规格书关键条目（重写版）

### 2.1 电压与频率适配参数

| 项目 | 重写条目 | 条款/来源 |
|---|---|---|
| 系统适用电网 | `日本九州地区低压配电网，标称频率 60.00Hz，三相三线式 200V 级配电体系` | `九州电力送配电` 智能电表相线/电压公开信息；`JET` 低压系統連系适用对象 |
| PCS 交流侧额定接口 | `AC 202.0V（三相三线式），60.00Hz` | `九州电力送配电` 三相三线式 200V 公网配置；计测精度表达按 `JIS C 8715-2:2019 §4` |
| 交流侧适用电压等级 | `交流输出电压 600V 以下` | `JIS C 4412-2:2019 §1` |
| 直流侧适用电压等级 | `直流入出力电压 750V 以下` | `JIS C 4412-2:2019 §1` |
| 电压测试计测允许差 | `202.0V 设定值下，测试与报告允许差为 ±0.5%` | `JIS C 8715-2:2019 §4 a)` |
| 电流测试计测允许差 | `额定充放电电流测试与报告允许差为 ±1.0%` | `JIS C 8715-2:2019 §4 b)` |
| 温度测试计测允许差 | `关键温升与保护动作测试的计测允许差为 ±2.0C` | `JIS C 8715-2:2019 §4 c)` |
| 时间测试计测允许差 | `保护动作延时、并网恢复等待时间等时间参数测试允许差为 ±0.1%` | `JIS C 8715-2:2019 §4 d)` |

### 2.2 推荐写法：测试条件

建议在规格书中直接写成：

> `交流侧额定接口：AC 202.0V, 3P3W, 60.00Hz（九州地区低压配电网适配）。型式试验与工厂受入试验中的电压、电流、温度及时间参数计测允许差分别按 JIS C 8715-2:2019 §4 a)~d) 执行。`

说明：

- 公开可核验的 `JIS C 8715-2:2019 §4` 明确给出了电压、电流、温度、时间等测试计测允许差。
- 公开可核验资料未直接给出 `频率计测允许差` 的 JIS 条文，因此规格书中宜把 `60.00Hz` 写成 `地区适配名义值`，并在 FAT/SAT 试验规程中单列频率源精度要求，不应伪造为 JIS 明文参数。

## 3. JIS C 8715 对应的 3 项核心安全条款

### 3.1 过充/过流/过温边界管理

**规格书写法：**

> `电池系统必须保证单电池始终运行在电芯制造商规定的温度、电压和电流范围内；PCS 与 BMS 联动控制逻辑不得使任何单体越过上限充电电压、最大充电电流及安全温度边界。`

**法规溯源：**

- `JIS C 8715-2:2019 §5.4 温度，電圧及び電流の管理`
- `JIS C 8715-2:2019 §8.2 バッテリーマネジメントシステム`
- `JIS C 8715-2:2019 §7.5.2 過充電試験`

**为什么是核心条款：**

- 公开可核验条文明确要求电池系统设计必须落在电芯制造商规定的温度、电压、电流范围内。
- 日本消防厅公开的住宅用蓄电池火灾验证，也将“过充电时不得发生火灾”列为核心验证项之一。

### 3.2 预见性误使用下的外部短路安全

**规格书写法：**

> `在可预见误使用条件下，包括外部短路工况，系统不得引发发火、破裂、因电解液漏出导致的持续短路，或导致内容物外露的开裂。`

**法规溯源：**

- `JIS C 8715-2:2019 §5.1 一般事項`
- `JIS C 8715-2:2019 §7.2 予見可能な誤使用`
- `JIS C 8715-2:2019 §7.2.1 外部短絡試験`

**为什么是核心条款：**

- `§5.1` 公开可核验文本已经明确列出了误使用后不得出现的 5 类潜在危险源。
- 日本消防厅公开验证项目中，第二项即为“单电池外部短路不得引发火灾”。

### 3.3 内部短路与类烧传播抑制

**规格书写法：**

> `系统设计必须对内部短路进行设计评价，并通过单体、模组、系统级隔离与保护设计，将热失控后果限制在局部，不得演变为外部类烧。`

**法规溯源：**

- `JIS C 8715-2:2019 §7.3 内部短絡に対する考慮−設計評価`
- `JIS C 8715-2:2019 §7.3.3 類焼試験（電池システム）`
- `JIS C 8715-2:2019 §8.1 一般要求`

**为什么是核心条款：**

- `§7.3` 是公开可核验的专门条款，直接对应内部短路设计评价。
- 日本消防厅公开验证项目中，第三项即为“内部短路发生时不得向系统外部类烧”。

## 4. 电网交互功能的本地化定义

### 4.1 本地化定义原则

`JIS C 4412-2:2019 §1` 已明确说明：

> `该标准不规定系統連系保护动作。`

因此，`PCS` 的并网保护、再并列、逆潮流处理、出力控制等功能，不应伪写成 `JIS C 4412` 的条款要求，而应按：

1. `电力品質確保に係る系統連系技術要件ガイドライン`
2. `JET 系統連系保護装置等認証`
3. `九州電力送配電` 的低压接入与出力控制接口要求

分别定义。

### 4.2 建议写入规格书的本地化条目

#### 1. 并网识别与切离

> `PCS 应适配日本九州地区 60Hz 低压配电网，具备并网状态识别、异常解列、停电时自立切换及复电后再并列等待功能。`

**来源：**

- `METI` 系統連系技术要件指南
- `JET` 系統連系保護装置等認証制度说明

#### 2. 再并列等待时间

> `复电后，PCS 应具备固定等待后再并列的控制逻辑；规格书初始推荐整定值为 300.0s，项目实际整定值以系統連系協議书为准。`

**来源：**

- `JET` 公开注册机型信息中，低压并网产品常见 `復電後一定時間の遮断装置投入阻止: 300秒`

#### 3. 再并列频率适配

> `系统应按西日本 60Hz 区域设置再并列频率判据；规格书初始推荐写法为“支持 60Hz 区域再并列频率判据，出厂整定示例 60.1Hz”，最终整定以接入审查结果为准。`

**来源：**

- `JET` 公开注册机型信息中可见 `再並列時の許容周波数対応（出荷時整定値 60.1Hz）`

#### 4. 电压上升抑制与逆潮流协调

> `系统应具备低压并网场景下的电压上升抑制及逆潮流协调功能；规格书初始推荐整定值可写为“出力制御起始电压 109.0V，允许整定范围 107.0V-113.0V”，最终应以九州地区接入审查和机型认证整定为准。`

**来源：**

- `JET` 公开注册机型信息中可见 `電圧上昇抑制機能 出力制御: 109V（107.0-113.0V）`
- `九州電力送配電` 智能电表与双方向计量公开说明确认了低压逆潮流计量场景存在

### 4.3 不应写错的本地化边界

1. 不应把 `50kW / 100kWh` 写成日本普通独栋住宅常规户储。
2. 不应把 `JIS C 4412` 直接写成“规定了所有并网保护整定值”。
3. 不应把 `JET` 注册示例整定值写成“全国统一法定固定值”。
4. 不应省略 `九州 = 60Hz` 的地区电网前提。

## 5. 可直接放入规格书的精简版本

```md
适用系统：日本九州地区低圧蓄電システム（60Hz）
系统规模：PCS 50.0kW / Battery 100.0kWh
系统拓扑：分离形 PCS + Lithium-ion battery system + BMS + EMS

1. 交流侧接口
- 电气方式：三相三线式
- 额定交流电压：AC 202.0V
- 额定频率：60.00Hz
- 适用区域：日本九州地区低压配电网
- 条款依据：JIS C 4412-2:2019 §1；JIS C 8715-2:2019 §4

2. 安全边界
- 电池系统必须保证单电池在制造商规定的温度、电压及电流范围内运行。
- 条款依据：JIS C 8715-2:2019 §5.4、§8.2
- 试验条款：JIS C 8715-2:2019 §7.5.2

3. 可预见误使用安全
- 外部短路及其他可预见误使用后，不得引发发火、破裂、持续性可燃气体喷出或内容物外露。
- 条款依据：JIS C 8715-2:2019 §5.1、§7.2
- 试验条款：JIS C 8715-2:2019 §7.2.1

4. 内部短路传播抑制
- 系统必须实施内部短路设计评价，并通过模组/系统级隔离设计抑制类烧扩展。
- 条款依据：JIS C 8715-2:2019 §7.3、§8.1
- 试验条款：JIS C 8715-2:2019 §7.3.3

5. 系统连系本地化
- 具备异常解列、停电自立、复电后再并列等待、逆潮流协调及电压上升抑制功能。
- 再并列等待时间初始推荐值：300.0s
- 再并列频率初始推荐值：60.1Hz 对应 60Hz 区域
- 出力控制起始电压初始推荐值：109.0V
- 条款依据：JET 系統連系保護装置等認証公开注册示例；METI 系統連系技術要件ガイドライン
```

## 6. 当前交付边界

本稿已经满足以下要求：

- 已按 `九州 60Hz` 场景重写关键规格条目。
- 已给出 `JIS C 8715-2` 的 3 项核心安全条款及对应条款号。
- 已将 `PCS` 的本地化并网交互功能从 `JIS` 与 `系統連系` 两条线分开定义。
- 已将公开可核验的计测允许差写到 `±0.5% / ±1.0% / ±2.0C / ±0.1%` 的精度级别。

仍需注意：

- `JIS C 4412:2021` 的全文属于受版权保护标准，公开页面能稳定核验到的是适用范围、目录结构和部分前言说明，不能把未公开的细分条文内容伪写成已核验条款。
- `频率计测允许差` 与部分并网整定值，在当前公开资料中更适合作为 `JET 注册示例值 / 项目整定初始值`，不应写成 `JIS` 明文固定值。
