## 产品分型优先于认证判断

| 类型 | 日本常见叫法 | 核心判断 |
|---|---|---|
| A | `ポータブル電源` | AC 输出便携储能，先判断是否电安法对象外，再看 S-JET 等安全表达 |
| B | `モバイルバッテリー` | DC 外接供电产品，通常重点看圆形 PSE |
| C | `低圧蓄電システム` | 低压并网型户储/小工商，重点看 JIS C 4412、S-JET、系统连系 |
| D | `定置用大型蓄電システム` | 大型定置储能，重点看 JIS C 4441、IEC 62933-5-2、NITE 等 |

## 必须长期跟踪的主题

1. PSE 对象/非对象判断。
2. METI 对便携式储能的安全要求。
3. JET S-JET 认证。
4. JIS C 4412 低压储能系统。
5. JIS C 4441 大型定置储能系统。
6. JET 系统连系保护装置等认证。
7. JC-STAR 对带通信储能系统的影响。
8. NITE 公共调达和重要基础设施安全指南。
