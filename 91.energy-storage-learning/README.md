## 使用原则

1. `agents/` 只定义角色和教学方式，不塞学习计划。
2. `roadmap/` 定义学什么、按什么顺序学、每周交付什么。
3. `learning/` 记录学习过程，允许不完美，但必须可追踪。
4. `research/` 记录证据、来源和待验证项，结论与证据分开。
5. `project/` 只放能转化为业务决策的成果。
6. 每周至少做一次 [Weekly Review](reviews/weekly-review-template.md)。

详见 [roadmap/roadmap.md](roadmap/roadmap.md)。

## 每次学习 Session 的推荐上下文

在 ChatGPT、Codex 或其他 Agent 中开启学习时，建议按顺序提供：

1. [agents/mentor.md](agents/mentor.md)
2. [roadmap/roadmap.md](roadmap/roadmap.md)
3. 当前周计划，例如 [roadmap/week01.md](roadmap/week01.md)
4. 最近一次周复盘或当前作业

不要每次复制所有资料。只加载当前任务需要的最小上下文。
