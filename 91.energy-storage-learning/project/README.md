# Project

本目录保存可以转化为业务决策的项目成果。

当前主项目：

- [japan-storage-business/](japan-storage-business/README.md)

学习材料进入 `learning/`，证据研究进入 `research/`。只有能服务业务判断的内容才进入 `project/`。

## 项目成果标准

项目文件必须回答：

1. 要做什么市场？
2. 为什么这个市场值得做？
3. 目标客户是谁？
4. 产品定义是什么？
5. 认证和合规路线是什么？
6. 中国供应链如何支撑？
7. 渠道如何进入？
8. 主要风险和兜底方式是什么？
9. 下一步投入需要什么资源？
